    static : define propriedades ou metodos que pertencem a classe em si e não a instancias individuais.
        exemplo de código e ultiliza um metodo estático e um não estático.
           
           
            class Carro {
                constructor(marca) {
                    this.marca = marca;
                }

                acelerar() {
                    console.log(this.marca + ' está acelerando!');
                }

                static ligarCarro() {
                    console.log('O carro está ligado!');
                }
                }

                const meuCarro = new Carro('Toyota');
                meuCarro.acelerar(); // Saída: Toyota está acelerando!

                Carro.ligarCarro(); // Saída: O carro está ligado!

    assync: torna a operação assíncrona, o que significa que podemos executar outras operações enquanto aquela não fica pronta. 
        seguindo a mesma lógica, _await_ é usada para aguardar o resultado de uma outra função assíncrona para aí sim executar
         outra função.
